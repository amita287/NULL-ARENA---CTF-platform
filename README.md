🛡️ NULL ARENA — CTF Platform

A full-stack Capture The Flag (CTF) platform with real-time monitoring, authentication, and security-focused design.

🚀 Overview

NULL ARENA is a web-based cybersecurity training platform designed to host and manage CTF challenges. It enables users to practice real-world attack and defense scenarios while providing administrators with monitoring and control capabilities.

CTF platforms are widely used for hands-on cybersecurity learning and competitions because they simulate real-world vulnerabilities and attacks.

✨ Features
🔐 User Authentication
Email/password login
Google OAuth integration

🧠 CTF Challenge System
Flag submission & validation
Score tracking (extendable)

📊 Monitoring & Observability
Prometheus metrics collection

🐳 Dockerized Deployment
Easy setup using Docker Compose
Multi-container architecture

⚡ Lightweight Backend
Flask-based application
SQLite database

Tech Stack
Backend: Flask (Python)
Frontend: HTML, CSS, JavaScript
Database: SQLite
Monitoring: Prometheus
Deployment: Docker, Docker Compose
Authentication: Google OAuth
